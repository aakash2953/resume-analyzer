# Resume Job Match Analyzer

A Streamlit web app that compares resumes with job descriptions using NLP.

## Features
- TF-IDF + Cosine Similarity
- PDF Resume Parsing
- Match Score Visualization

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- NLTK
- Matplotlib

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
